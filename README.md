### 📌 Project Summary

An end-to-end Machine Learning project that predicts whether an engineering fresher is likely to be selected during the recruitment process. The project includes Exploratory Data Analysis (EDA), Logistic Regression model development, performance evaluation, business insights, and deployment through a Streamlit web application.

### 🎯 Key Highlights

* Built using Python, Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn, and Streamlit
* Trained on 2,000 fresher candidate records
* Interactive web-based prediction system
* Complete hiring analytics dashboard
* Feature importance analysis
* Business insight generation
* Deployment-ready architecture

### 💼 Business Problem

Recruiters often receive thousands of applications from fresher candidates. Manually reviewing every profile is time-consuming and inconsistent. This project uses Machine Learning to assist recruiters by identifying candidates who are more likely to be shortlisted based on academic performance, technical skills, projects, certifications, internship experience, and assessments.

### 🧠 Machine Learning Approach

Algorithm Used:

* Logistic Regression

Target Variable:

* Selected = 1
* Not Selected = 0

Input Features:

* Age
* CGPA
* Aptitude Score
* Programming Score
* SQL Score
* Communication Score
* Projects
* Certifications
* Internship Months
* Hackathon Participation
* Attendance
